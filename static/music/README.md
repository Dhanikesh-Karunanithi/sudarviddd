# Bundled background loops

The `.wav` files here are **short synthetic stereo sine tones** generated by `scripts/bake_music_loops.py` (stdlib `wave` only) for use as royalty-free background beds. They are not third-party musical works.

Regenerate:

```bash
python scripts/bake_music_loops.py
```

`ffmpeg` muxes them into `audio/music.mp3` per job when `include_music` is true. Theme → file mapping lives in `sudarvid/media.py` (`THEME_MUSIC_MAP`).
